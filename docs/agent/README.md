# Agent Working Docs

This directory holds durable project memory that should travel with the repo.

Use these files for current SEO/ASO status, product marketing context, directory
submissions, and the latest tracked audit artifacts. Keep transient plans,
private notes, local skills, and archived scratch work in ignored `.agents/`
instead.

Current priority files:

- `product-marketing-context.md` — positioning, ICP, objections, customer language, and brand voice.
- `directory-profile.md` — public-safe listing and directory submission facts.
- `seo-progress.md` — first stop for SEO/AIO work.
- `aso-appstore-listing.md` and `play-store-listing.md` — store-listing working copy and status.
- `seo-audit-2026-04-20/` — latest tracked audit bundle.
